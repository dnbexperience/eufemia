#Change log of Eufemia

You may also be interested in details about releases and have a look at the GitHub Releases for versioning of the @dnb/eufemia.

#May, 1. 2026

  • Deprecated the --color-* CSS custom properties. Use Design Tokens – Colors instead.

#April, 21. 2026

  • New major version 11
  • InputMasked and all related masked input fields have been rewritten with a new appearance.
  • DatePicker and Field.Expiry have changed interaction behavior and no longer support a real cursor.
  • Card: adjusted outline thickness and border radius to new design.
  • List: added List.ShowMoreButton for expandable lists.
  • Button: removed variant="signal". Use variant="primary" or variant="secondary" instead.
  • Forms: removed default outset from Form.Card, Wizard, and inline help; removed heading auto-indent.
  • Forms: removed included Ajv instance creation from Form.Handler and fields.
  • New component: Menu for accessible dropdown menus with keyboard navigation and nested menu support.
  • New focus color for components when using design tokens.

#February, 27. 2026

  • New component: Stat, including Stat.Currency, Stat.Percent, Stat.Trend, and Stat.Label for prominent metric formatting.

#January, 30. 2020

#New DNB font

General information: All existing and new components uses the new (and slightly larger) text styles, but the existing height of the components is not affected by this and has NOT changed.

Typography changes:

  • h1 DNB Sans Medium 48px / 56px line height
  • h1 small DNB Sans Medium 34px / 40px line height
  • h2 DNB Sans Medium 26px / 32px line height
  • h3 / Lead DNB Sans Medium 20px / 28px line height
  • Body text DNB Sans Regular 18px / 24px line height
  • Body text medium DNB Sans Medium 18px / 24px line height
  • Small text DNB Sans Regular 16px / 20px line height
  • Small text medium DNB Sans Medium 16px / 20px line height
  • X-Small text DNB Sans Regular 14px / 16px line height
  • X-Small text medium DNB Sans Medium 14px / 16px line height

Number features are removed as we do not need them anymore.
