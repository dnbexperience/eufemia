#Primary Icons

The Primary Icons are used inside Eufemia components and are also bundled as a UMD Package.

Primary Icons can easily be defined and included in components, for instance, in Buttons. They are integrated by using the IconPrimary component.

#React example usage

<IconPrimary icon="bell" size="medium" />
<Button icon="chevron_right" />

You can also import them like the Secondary icons:

import { bell_medium, chevron_right } from '@dnb/eufemia/icons'


<Icon icon={bell_medium} size="medium" />
<Button icon={chevron_right} />
