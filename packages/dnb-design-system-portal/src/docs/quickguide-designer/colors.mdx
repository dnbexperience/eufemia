## Gradients, shadows, and shades

### Gradients

The DNB brand consists mainly of solid fill colors. Gradients should be avoided despite there being an illustration in Brandbook on page 39 (pdf) showing a graph with a purple gradient.

### Shadows

Shadows are used to depict depth along the z-axis. Typical use cases are interface components that are laid 'above' the main interface such as calendar widgets. Currently, Android development for DNB uses Google Material Design Guidelines regarding shadow color, depth, blur, etc.

See the DNB Figma main guide for shadow specifications.

## Using Colors in Code

You can use the defined colors in your code as follows:

```jsx
import { colors } from '@dnb/eufemia/shared'

// Example using colors from DNB UI
const dnbColors = colors.ui

const MyComponent = () => {
  return (
    <div>
      Hex values:
      <div>
        <Logo color={dnbColors.oceanGreen.hex} size="100" />
        <Logo color={dnbColors.emeraldGreen.hex} size="100" />
        <Logo color={dnbColors.seaGreen.hex} size="100" />
        <Logo color={dnbColors.mintGreen.hex} size="100" />
        <Logo color={dnbColors.summerGreen.hex} size="100" />
        <Logo color={dnbColors.accentYellow.hex} size="100" />
        <Logo color={dnbColors.indigo.hex} size="100" />
        <Logo color={dnbColors.violet.hex} size="100" />
      </div>
      Rgb values:
      <div>
        <Logo color={dnbColors.oceanGreen.rgb} size="100" />
        <Logo color={dnbColors.emeraldGreen.rgb} size="100" />
        <Logo color={dnbColors.seaGreen.rgb} size="100" />
        <Logo color={dnbColors.mintGreen.rgb} size="100" />
        <Logo color={dnbColors.summerGreen.rgb} size="100" />
        <Logo color={dnbColors.accentYellow.rgb} size="100" />
        <Logo color={dnbColors.indigo.rgb} size="100" />
        <Logo color={dnbColors.violet.rgb} size="100" />
      </div>
    </div>
  )
}
```
