# Logos

## Brand Center

**NB!** If you have access to the DNB's Brand Center, [you can find the logos under "Visuell identitet"](https://bc.dnb.no/brandcenter/no/dnbvisualguidelines/component/default/761).

## Brandbook

From the Brandbook PDF file, digital media should use the 3rd level logos (page 9).

## Developers

The DNB Logo can easily be used and [integrated into digital web products as a component](/uilib/components/logo).
