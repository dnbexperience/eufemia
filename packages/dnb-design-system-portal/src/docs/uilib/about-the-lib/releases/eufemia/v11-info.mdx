## Install

To upgrade to @dnb/eufemia v11 with NPM, use:

```bash
$ npm i @dnb/eufemia@11
# or
$ yarn add @dnb/eufemia@11
```

## Section

1. Most of the `style_type` variants were removed. Instead use the new `variant` property:

- `mint-green`
- `sea-green`
- `emerald-green`
- `lavender`
- `black-3`
- `sand-yellow`
- `pistachio`
- `fire-red`
- `fire-red-8`
