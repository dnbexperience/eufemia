## Install

To upgrade to @dnb/eufemia v12 with NPM, use:

```bash
$ npm i @dnb/eufemia@12
# or
$ yarn add @dnb/eufemia@12
```

## Component changes

### [Modal](/uilib/components/modal/) and [Drawer](/uilib/components/drawer/)

- Replace `closeButtonAttributes` with `closeButtonProps`.

```diff
-<Drawer closeButtonAttributes={{ text: 'Custom text' }} />
+<Drawer closeButtonProps={{ text: 'Custom text' }} />
```

### [InfoCard](/uilib/components/info-card/)

- Replace `closeButtonAttributes` with `closeButtonProps`.

```diff
-<InfoCard closeButtonAttributes={{ href: '/path' }} />
+<InfoCard closeButtonProps={{ href: '/path' }} />
```
