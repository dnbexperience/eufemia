### Basic usage

The basic use of `<Anchor>` is identical to the `<a>` tag.

<AnchorBasicUse />

### With icon

Icons can be added with the `icon` and `iconPosition` props. Normally by sending in the name if an icon, but it is also possible to send in html/react code (normally for custom svg).

<VisibilityByTheme visible="sbanken">
  For Sbanken, links with icons have a slightly different styling that is
  not intended to be used in a regular paragraph. The class
  `.dnb-anchor--inline` can be used to force default styling even if there
  is an icon.
</VisibilityByTheme>
