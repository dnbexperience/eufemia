### Corner badge

You can "float" the badge to the corner of an element by wrapping the `<Badge>` component around it.

<BadgeInformationAvatar />
<BadgeNotificationAvatar />

#### Setting property `horizontal` and `vertical`

When placing the badge in the corner you can control its position.

<BadgeCornerPosition />

### Setting the `status` and `subtle` properties

The information variant has 5 possible `status` values, and two possible `subtle` values.

The default state is equivalent to `status='default'` and `subtle={false}`.

<BadgeStatus />

### Hiding Badge with `hideBadge`

Sometimes you need to hide the badge without hiding the wrapped content. To make this less complicated you can use the `hideBadge` prop.

The example below hides the badge when there are no notifications. You can add or remove notifications with the "+" and "-" buttons.

<BadgeHide />
