### Icon button

When the button has no text an `aria-label` attribute is required for accessibility. And a `title` attribute is recommended for mouse hover.

We automatically add an `aria-label` with the same value as `title` if `text` and `children` properties are undefined. So in most cases you only have to set the `title` property.

### Vipps Wallet Button

To import `VippsWalletButton` by path, including its extra styles:

```tsx
import VippsWalletButton from '@dnb/eufemia/components/button/custom/vipps-wallet-button/VippsWalletButton'
import '@dnb/eufemia/components/button/custom/vipps-wallet-button/style/dnb-vipps-wallet-button.min.css'

// For Style Isolation, use: dnb-vipps-wallet-button--isolated.min.css
```

The label text is translated internally and follows `Provider` locale: `nb-NO`: `Legg til i`, `en-GB`: `Add to`, `sv-SE`: `Lägg till i`, `da-DK`: `Tilføj til`.
