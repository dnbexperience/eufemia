## Description

A ready to use DNB date formatter. Use it wherever you want to format dates.

Good reasons to use this component:

- makes the date formatting uniform for all DNB applications.
- makes dates accessible to screen readers.

The component is using `Intl.DateTimeFormat` browser API, and supports the following options.

- `dateStyle`
- `weekday`
- `day`
- `month`
- `year`

More detailed documentation of the options can be found [here](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/DateTimeFormat/DateTimeFormat#date-time_component_options)
