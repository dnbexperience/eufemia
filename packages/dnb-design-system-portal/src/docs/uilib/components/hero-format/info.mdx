## Import

```tsx
import { HeroFormat } from '@dnb/eufemia'
```

## Description

`HeroFormat` contains formatting components for prominent values, where typography and visual emphasis are part of the component API.

Use `HeroFormat.Amount` for currency amounts shown in hero sections, summaries, and key number highlights.

## Basic usage

```tsx
import { HeroFormat } from '@dnb/eufemia'

render(<HeroFormat.Amount value={123} />)
```

This namespace pattern keeps the API scalable and tree-shakeable for additional formatters.

## Currency Amount

`HeroFormat.Amount` is built on the [NumberFormat](/uilib/components/number-format/) formatting logic.

It adds typography-specific props:

- `mainSize`
- `auxiliarySize`

## Accessibility

`HeroFormat.Amount` keeps a dedicated screen-reader value (`.dnb-sr-only`) based on the localized formatted number.

Use `srLabel` to prepend context, for example what the amount represents.

When e.g. `signDisplay="always"` is used, the sign is rendered as a separate visual element with CSS spacing, while the accessible text stays based on the formatted number string.
