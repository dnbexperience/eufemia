## Description

The InfoCard is used to give the user more information than we can give in a [FormStatus](/uilib/components/form-status/) (messagebox). It can also be used to give useful tips.

The text content is set to a max width of 70 characters to ensure an optimal [UU](https://www.uutilsynet.no/veiledning/tekst-og-struktur/226) recommended reading lengths.
