## Number mask properties

The number mask is used for all kinds of number based masks, like:

**Locale based masks:**

- `asNumber`
- `asCurrency`
- `asPercent`

**Static masks:**

- `numberMask`
- `currencyMask`

You can `maskOptions` to manipulate the options.

Defaults to Norwegian number format.
