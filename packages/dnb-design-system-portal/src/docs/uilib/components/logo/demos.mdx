## Demos

### Importing a logo

To use a logo, the svg must be imported and handed to the `Logo` components throught the `svg` prop.

```jsx
import {
  DnbDefault,
  CarnegieDefault,
  EiendomDefault,
  SbankenDefault,
  SbankenHorizontal,
  SbankenCompact,
} from '@dnb/eufemia/components/Logo'
```

<LogoAllExample />

If no svg is provided, the `DnbDefault` logo is used by default:

<LogoDefaultExample />

### Change logo based on theme

The `svg` prop can also accept a function that returns a logo svg based on the current [theme](/uilib/usage/customisation/theming/theme/) props.

```tsx
import type { ThemeProps } from '@dnb/eufemia/shared/Theme'
```
