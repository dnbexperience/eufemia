## `Radio` events

| Events      | Description                                                                                                   |
| ----------- | ------------------------------------------------------------------------------------------------------------- |
| `on_change` | _(optional)_ will be called on state changes made by the user. Returns an object `{ checked, value, event }`. |

## `Radio.Group` events

| Events      | Description                                                                                              |
| ----------- | -------------------------------------------------------------------------------------------------------- |
| `on_change` | _(optional)_ will be called once a Radio button changes the state. Returns an object `{ value, event }`. |
