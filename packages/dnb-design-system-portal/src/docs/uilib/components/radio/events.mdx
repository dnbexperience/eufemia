## Radio button events

| Events      | Description                                                                                                               |
| ----------- | ------------------------------------------------------------------------------------------------------------------------- |
| `on_change` | _(optional)_ will be called on state changes made by the user. Returns a boolean and string `{ checked, value, event }`. |

## Radio events

| Events      | Description                                                                                              |
