## Import

```tsx
import { Spotlight } from '@dnb/eufemia'
```

## Description

`Spotlight` contains formatting components for prominent values, where typography and visual emphasis are part of the component API.

Use `Spotlight.Currency` for currency amounts and `Spotlight.Percent` for percentage highlights in hero sections, summaries, and key numbers.

## Basic usage

```tsx
import { Spotlight } from '@dnb/eufemia'

render(<Spotlight.Currency value={123} />)
```

This namespace pattern keeps the API scalable and tree-shakeable for additional formatters.

## Currency and Percent

`Spotlight.Currency` and `Spotlight.Percent` are built on the [NumberFormat](/uilib/components/number-format/) formatting logic.

It adds typography-specific props:

- `mainSize`
- `auxiliarySize`

## Accessibility

Both components keep a dedicated screen-reader value (`.dnb-sr-only`) based on the localized formatted number.

Use `srLabel` to prepend context, for example what the amount represents.

When e.g. `signDisplay="always"` is used, the sign is rendered as a separate visual element with CSS spacing, while the accessible text stays based on the formatted number string.
