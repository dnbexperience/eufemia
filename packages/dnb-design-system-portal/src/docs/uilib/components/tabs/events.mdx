### Prevent a change

You can prevent a change from happening by returning false on the `on_click` event handler:

```tsx
<Tabs
  on_click={() => {
    if (condition === true) {
      return false
    }
  }}
  on_change={() => {
    // Will not get emitted
  }}
/>
```
