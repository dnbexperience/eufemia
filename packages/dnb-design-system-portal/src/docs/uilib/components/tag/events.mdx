## `Tag` events

| Events     | Description                                                                                                                                                                         |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `onClick`  | _(optional)_ will be called on a click event. Returns an object with the native event: `{ event }`.                                                                                 |
| `onDelete` | _(optional)_ will be called on a click event. Returns an object with the native event: `{ event }`. This property does not support icon and will be ignored if `onClick`is defined. |
