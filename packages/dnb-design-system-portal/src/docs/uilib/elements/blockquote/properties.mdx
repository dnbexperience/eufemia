## Properties

| Properties     | Description                                                                                                                                        |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| `noBackground` | _(optional)_ Hides the blockquote background by making it transperant.                                                                             |
| `direction`    | _(optional)_ Determines the flow direction of the content inside of blockquote. Can be either `horizontal` or `vertical`. Defaults to `horizontal` |
