# Paragraph

## Paragraph class modifiers

Eufemia comes with several styles you can use on paragraphs and other HTML text elements:

**Weights**

- `.dnb-p` (Body text)
- `.dnb-p--medium`

**Sizes**

- `.dnb-p--small`
- `.dnb-p--x-small`

**Variants**

- `.dnb-p--lead`
