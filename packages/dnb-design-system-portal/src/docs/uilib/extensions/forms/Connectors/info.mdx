## Description

`Connectors` are an opt-in way to extend the functionality of a form. They can be used to add features like API calls for autocomplete, validation, and more.

Current connectors:

- [Bring](/uilib/extensions/forms/Connectors/Bring/)

## Import

```tsx
import { Connectors } from '@dnb/eufemia/extensions/forms'
```
