## Outset

`<Form.Card>` sets `outset` by default to perserve content alignement in forms.
If for some reason this breaks layout `outset={false}` can be used to position
the `<Form.Card>` as a regular `<Card>` component.

Alternatively, the wrapper `<Form.Card.Provider hasParentCard>` can be used to have children behave as if they are inside a `<Card>` component.
