## Description

`Iterate.RemoveButton` connects to the array of a surrounding `Iterate.Array` and removes the item when clicked.

```tsx
import { Iterate, Field } from '@dnb/eufemia/extensions/forms'

render(
  <>
    <Iterate.Array path="/">
      <Field.Name.Last itemPath="/name" />
    </Iterate.Array>

    <Iterate.RemoveButton text="Remove item" path="/" pushValue={{}} />
  </>,
)
```

## Confirm removal

You can adding the `confirmRemove` property open a confirmation dialog before removing the item.

```tsx
<Iterate.RemoveButton confirmRemove />
```
