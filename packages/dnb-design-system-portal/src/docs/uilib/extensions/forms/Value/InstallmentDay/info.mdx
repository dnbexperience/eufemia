## Import

```tsx
import { Value } from '@dnb/eufemia/extensions/forms'
render(<Value.InstallmentDay />)
```

## Description

`Value.InstallmentDay` displays the selected installment day as a read-only value. When the value is `'last'`, it displays the translated "Last day of month" label.

There is a corresponding [Field.InstallmentDay](/uilib/extensions/forms/feature-fields/InstallmentDay) component.
