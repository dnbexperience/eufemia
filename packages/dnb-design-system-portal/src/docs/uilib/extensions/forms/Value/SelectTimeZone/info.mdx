## Import

```tsx
import { Value } from '@dnb/eufemia/extensions/forms'
render(<Value.SelectTimeZone />)
```

## Description

`Value.SelectTimeZone` will render the selected time zone name by `value`'s IANA time zone identifier (e.g., `Europe/Oslo`). It displays the time zone name in the current locale. If the value provided is not a valid/supported time zone identifier, it displays the value.

There is a corresponding [Field.SelectTimeZone](/uilib/extensions/forms/feature-fields/SelectTimeZone) component.

```jsx
import { Value } from '@dnb/eufemia/extensions/forms'
render(<Value.SelectTimeZone path="/timezone" />)
```

### The `useTimeZone` hook

You can use the `Value.SelectTimeZone.useTimeZone` hook to get the time zone name by IANA time zone identifier. It returns the time zone name in the current locale.

```tsx
import { Value } from '@dnb/eufemia/extensions/forms'

const MyComponent = () => {
  const { getTimeZoneNameByIdentifier } = Value.SelectTimeZone.useTimeZone()
  const name = getTimeZoneNameByIdentifier('Europe/Oslo')
}
```
