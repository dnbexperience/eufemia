### Component-specific props

| Property                                | Type     | Description                                                                           |
| --------------------------------------- | -------- | ------------------------------------------------------------------------------------- |
| `trueText`                              | `string` | Text to show in the UI when value is `true`.                                          |
| `falseText`                             | `string` | Text to show in the UI when value is `false`.                                         |
