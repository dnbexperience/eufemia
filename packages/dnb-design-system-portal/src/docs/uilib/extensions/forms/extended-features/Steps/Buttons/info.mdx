## Description

`Steps.Buttons` Is a composition for a row containing both [PreviousButton](/uilib/extensions/forms/extended-features/Steps/PreviousButton/) and [NextButton](/uilib/extensions/forms/extended-features/Steps/NextButton/).

These two buttons are wrapped in the [Form.ButtonRow](/uilib/extensions/forms/extended-features/Form/ButtonRow/) component.

```jsx
import { Form, Steps } from '@dnb/eufemia/extensions/forms'

render(
  <Form.Handler>
    <Steps.Layout>
      <Steps.Step title="Step 1">
        <Steps.Buttons />
      </Steps.Step>
    </Steps.Layout>
  </Form.Handler>,
)
```
