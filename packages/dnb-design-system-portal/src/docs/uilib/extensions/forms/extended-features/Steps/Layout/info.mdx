## Description

`Steps.Layout` is a wrapper component for showing forms with a [StepIndicator](/uilib/components/step-indicator/) for navigation between several steps.

Use the [Step](/uilib/extensions/forms/extended-features/Steps/Step/) component to define the steps. It uses `Flex.Stack` so you don't need to wrap your content additionally.

```tsx
import { Form, Steps } from '@dnb/eufemia/extensions/forms'

const MyForm = () => {
  return (
    <Form.Handler>
      <Steps.Layout>
        <Steps.Step title="Step 1">...</Steps.Step>
      </Steps.Layout>
    </Form.Handler>
  )
}
```

You can also split or separate the `Steps.Step` and its content:

```tsx
import { Form, Steps } from '@dnb/eufemia/extensions/forms'

// You can use the `Steps.Step` in an external component like this:
const Step1 = () => (
  <Steps.Step title="Step 1">
    <Form.MainHeading>Heading</Form.MainHeading>
    <Card stack>
      <P>Contents</P>
    </Card>
    <Card stack>
      <P>Contents</P>
    </Card>
    <Steps.NextButton />
  </Steps.Step>
)

const MyForm = () => {
  return (
    <Form.Handler>
      <Steps.Layout>
        <Step1 />
      </Steps.Layout>
    </Form.Handler>
  )
}
```

You can mix and match the usage of `Steps.Step` and `Flex.Stack` depending on your needs:

```tsx
import { Form, Steps } from '@dnb/eufemia/extensions/forms'

const Step2 = () => (
  <Flex.Stack>
    <Card stack>
      <P>Contents</P>
    </Card>
    <Card stack>
      <P>Contents</P>
    </Card>
  </Flex.Stack>
)

const MyForm = () => {
  return (
    <Form.Handler>
      <Steps.Layout>
        <Steps.Step title="Step 2">
          <Form.MainHeading>Heading</Form.MainHeading>
          <Step2 />
          <Steps.Buttons />
        </Steps.Step>
      </Steps.Layout>
    </Form.Handler>
  )
}
```

## Controlling the steps

To define a different initial index (than 0), you can use `initialActiveIndex` hook.

For controlling the steps interactively, you can use the [useStep](/uilib/extensions/forms/extended-features/Steps/useStep/) hook:

```tsx
import { Form, Steps } from '@dnb/eufemia/extensions/forms'

const MyStep = () => {
  const { setActiveIndex, activeIndex } = Steps.useStep()
  return (
    <Card stack>
      <Button onClick={() => setActiveIndex(1)}>Go to step 2</Button>
    </Card>
  )
}

const MyForm = () => {
  return (
    <Form.Handler>
      <Steps.Layout initialActiveIndex={3}>
        <Steps.Step>
          <MyStep />
        </Steps.Step>
      </Steps.Layout>
    </Form.Handler>
  )
}
```

When using the `useStep` hook outside of the `Steps.Layout` context, you need to provide na unique `id` (string):

```tsx
import { Form, Steps } from '@dnb/eufemia/extensions/forms'

const MyForm = () => {
  const { setActiveIndex, activeIndex } = Steps.useStep('unique-id')

  return (
    <Form.Handler>
      <Steps.Layout id="unique-id">
        <Steps.Step>
          <Button onClick={() => setActiveIndex(0)}>Step 1</Button>
        </Steps.Step>
        <Steps.Step>
          <Button onClick={() => setActiveIndex(1)}>Step 2</Button>
        </Steps.Step>
      </Steps.Layout>
    </Form.Handler>
  )
}
```
