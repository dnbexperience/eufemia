## Description

`Steps.PreviousButton` connects to the `Steps.Context` to move the user to the previous step when clicked.

```jsx
import { Form, Steps } from '@dnb/eufemia/extensions/forms'

render(
  <Form.Handler>
    <Steps.Layout>
      <Steps.Step title="Step 1">
        <Steps.PreviousButton />
      </Steps.Step>
    </Steps.Layout>
  </Form.Handler>,
)
```
