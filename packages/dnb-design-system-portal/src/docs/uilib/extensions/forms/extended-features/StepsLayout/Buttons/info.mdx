## Description

`StepsLayout.Buttons` Is a composition for a row containing both PreviousButton and NextButton.

```jsx
import { StepsContext, StepsLayout } from '@dnb/eufemia/extensions/forms'
render(
  <StepsContext.Provider>
    <StepsLayout.Buttons />
  </StepsContext.Provider>,
)
```
