## Description

`StepsLayout.Step` shows child components when the surrounding [StepsLayout](/uilib/extensions/forms/extended-features/StepsLayout/) has been navigated to this step. StepsLayout keeps track of what is the active step, and navigating between steps is done through callbacks on the StepsContext, i.e. using [navigation buttons](/uilib/extensions/forms/extended-features/StepsLayout/Buttons/).

```jsx
import { StepsLayout, Form } from '@dnb/eufemia/extensions/forms'

const Step1 = () => {
  return (
    <StepsLayout.Step title="Step 1">
      <Form.MainHeading>Heading</Form.MainHeading>
      <Card>
        <P>Contents</P>
      </Card>
      <Card>
        <P>Contents</P>
      </Card>
      <StepsLayout.Buttons />
    </StepsLayout.Step>
  )
}

const MyForm = () => {
  return (
    <Form.Handler>
      <StepsLayout>
        <Step1 />
      </StepsLayout>
    </Form.Handler>
  )
}
```
