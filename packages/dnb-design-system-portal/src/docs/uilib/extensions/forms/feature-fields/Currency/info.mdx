## Description

`Field.Currency` is a wrapper component for the [input of numbers](/uilib/extensions/forms/base-fields/Number), with user experience tailored for currency values.

```jsx
import { Field } from '@dnb/eufemia/extensions/forms'
render(<Field.Currency />)
```

There is a corresponding [Value.Currency](/uilib/extensions/forms/Value/Currency) component.
