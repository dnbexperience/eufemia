## Description

`Field.Date` is a wrapper component for the [input of strings](/uilib/extensions/forms/base-fields/String), with user experience tailored for date values.

```jsx
import { Field } from '@dnb/eufemia/extensions/forms'
render(<Field.Date />)
```

There is a corresponding [Value.Date](/uilib/extensions/forms/Value/Date) component.
