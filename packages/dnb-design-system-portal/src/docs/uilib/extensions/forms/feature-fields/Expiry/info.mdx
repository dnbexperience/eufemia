## Description

`Field.Expiry` is a wrapper component for the input of strings, with user experience tailored for national identity number values.

```jsx
import { Field } from '@dnb/eufemia/extensions/forms'
render(<Field.Expiry />)
```

There is a corresponding [Value.Expiry](/uilib/extensions/forms/create-component/Value/Expiry) component.
