### Transform to a backend format

Internally the field works with clean values – a number (`1`–`28`) or `'last'`. When the backend expects a different wire format, such as a fixed two-digit day (`'01'`–`'28'`) with a sentinel for the last day of the month, use `transformOut` to convert the value at the API boundary. The converted value is what gets stored in the form data and passed to `onChange` and `onSubmit`. Use `transformIn` to convert incoming data (for example initial values) back to the field's internal value so it displays correctly.
