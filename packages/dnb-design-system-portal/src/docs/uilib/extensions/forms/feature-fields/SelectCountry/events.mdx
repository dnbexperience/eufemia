### Argument value

The event handlers has two arguments. The first one is a `string` containing the `ISO` of the selected countr, e.g. `CH`, and the second argument is an object with the properties `cdc`, `continent`, `i18n` and `iso`. e.g. `{
  cdc: '41',
  iso: 'CH',
    i18n: {
    en: 'Switzerland',
    nb: 'Sveits'
  },
  continent: 'Europe'
}`
