## Description

The PaymentCard component is a dynamic defined visual component, that imitates a physical payment card, having the equal size as the physical payment card in `mm`.
It exists as an independent extension to Eufemia.

How to use it:

1. First, define your desired look and design.
2. And import and include it in your application:

```jsx
import '@dnb/eufemia/extensions/payment-card/style'
import PaymentCard, {
  getCardData,
} from '@dnb/eufemia/extensions/payment-card'

render(<PaymentCard product_code="..." />)
```
