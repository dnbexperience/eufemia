## Import

```tsx
import '@dnb/eufemia/extensions/payment-card/style' // use "/style/isolated" for isolated styles
import PaymentCard, {
  getCardData,
} from '@dnb/eufemia/extensions/payment-card'

render(<PaymentCard productCode="..." />)
```

### Alternative CSS import

Some bundler configurations (e.g. certain Webpack setups) may not resolve the JavaScript-based style import above. If the styles are not being applied, you can import the pre-built CSS file directly instead:

```tsx
import '@dnb/eufemia/extensions/payment-card/style/dnb-payment-card.min.css'
```

Or if you are using [isolated styles](/uilib/usage/customisation/styling/style-isolation):

```tsx
import '@dnb/eufemia/extensions/payment-card/style/dnb-payment-card--isolated.min.css'
```

## Description

The PaymentCard component is a dynamically defined visual component that imitates a physical payment card. It exists as an independent extension to Eufemia.
