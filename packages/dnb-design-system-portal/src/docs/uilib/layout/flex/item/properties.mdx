## Properties

| Property                                | Type                        | Description                                                                                                                                                                                                                                                                                                                                        |
| --------------------------------------- | --------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `grow`                                  | `boolean`                   | _(optional)_ True to expand in width/height when there is more space available.                                                                                                                                                                                                                                                                    |
| `shrink`                                | `boolean`                   | _(optional)_ True to shrink in width/height when there is not enough space available for all components within the container.                                                                                                                                                                                                                      |
| `size`                                  | `object` or `number`        | _(optional)_ To set the size (parts) in "percentage" with numbers from 1 to 12 (`sizeCount`). You can also provide [Media Query](/uilib/usage/layout/media-queries/) types in an object. You can also use the value `auto` to disable it on a specific screen size. Wrap your Flex.Item's inside a [Flex.Container](/uilib/layout/flex/container). |
| `element`                               | `string` or `React.Element` | _(optional)_ Define the type of element. Defaults to `div`.                                                                                                                                                                                                                                                                                        |
| `innerRef`                              | `React.Ref`                 | _(optional)_ Provide a React.Ref to accessing the inner HTML element.                                                                                                                                                                                                                                                                              |
| [Space](/uilib/layout/space/properties) | Various                     | _(optional)_ spacing properties like `top` or `bottom` are supported.                                                                                                                                                                                                                                                                              |
