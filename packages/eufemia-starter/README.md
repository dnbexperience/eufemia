## Removing unused Eufemia CSS

Run a production build that strips component CSS this app does not use:

```bash
yarn build:optimized
```

This runs `vite build`, then post-processes the emitted CSS with PurgeCSS using
a safelist generated by `createSafelist` from the Eufemia style manifest (see
[scripts/purge-css.mts](scripts/purge-css.mts)). The console prints the detected
components and the before/after CSS size so you can verify the result.
