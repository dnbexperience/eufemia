# Fragments

## Import

You import them like so:

```jsx
import {
  DrawerList,
  ScrollView,
  TextCounter,
} from '@dnb/eufemia/fragments'
```

## Description

Fragments are small, low-level and reusable parts used inside other components.

You may use them only to build new components from.
